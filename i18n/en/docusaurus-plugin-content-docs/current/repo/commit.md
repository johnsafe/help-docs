开发人员可以查看每次提交代码的历史列表。

进入代码库详情页面，点击提交菜单。根据实际情况可以选择分支过滤、支持按照提交 Message 进行检索。

![](./img/25.jpg)

点击条目标题或提交 ID 可以查看提交详情：
![](./img/26.jpg)

针对提交详情支持评论，评论内容将通过邮件通知提交人：
![](./img/27.jpg)
